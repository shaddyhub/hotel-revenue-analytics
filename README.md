# 🏨 Hotel Revenue Analytics

> Exploring booking trends, revenue drivers, and cancellation patterns across a hotel portfolio.

## 📌 Project Overview
This project analyses hotel booking data to uncover what drives revenue performance — and where money is being lost through cancellations and low-value channels.

## 🔍 Key Questions Answered
- Which months generate the most revenue?
- Do City Hotels or Resort Hotels perform better?
- Which source markets are most valuable?
- Which distribution channels have the highest cancellation rates?

## 📊 Key Insights
- Summer months (Jul–Aug) consistently peak in revenue
- Resort Hotels command a higher ADR than City Hotels
- Online Travel Agents show the highest cancellation rates
- Spain and Germany are the top two source markets

2. Open `notebooks/hotel_revenue_analysis.ipynb` in Jupyter
